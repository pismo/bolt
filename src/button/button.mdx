# Button

Os botões são disponibilizados por uma classes principal: **btn** e outra classe que define o estilo de cores e formato, exemplo: **btn-primary**
A classe pode ser adicionada em qualquer elemento, mas é recomendado ser adicionada ao elemento button, assim garante as funcionalidades de _focus_ e _disabled_
