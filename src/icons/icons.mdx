# Icons

> Nota: ícones multicores, como é o caso dos cartões, são divididos em partes. Então sua composição segue o padrão a seguir:

```jsx
<div className="tw-i-master">
  <span className="path1" />
  <span className="path2" />
  <span className="path3" />
  <span className="path4" />
</div>
```
