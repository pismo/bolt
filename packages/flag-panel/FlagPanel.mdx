# FlagPanel

__FlagPanel__ fornece um Panel list com ferramentas para mostrar um menu de linguagens

### Propriedades

O component __FlagPanel__ chama a API [Country Flags](https://www.countryflags.io/)

| **prop**      | **type**                             | **required**   |   **desc**                                           |
| ------------- | ------------------------------------ | :--------------: | ---------------------------------------------------- |
| title         | string                               | true           | Valor que aparece quando o painel está fechado         |
| data          | {flag: FLAG, description: string}[]  | true           | array de objetos que representa a lista de idiomas contendo a bandeira (_flag_) e o título do idioma (_description_) |
| value         | {flag: FLAG, description: string}    | true           | item selecionado   |
| size          | `16 | 24 | 32 | 48 | 64`             | false (default 32) | tamanho da bandeira |
| type          | `'flat' | 'shiny'`                   | false (default 'shiny') | estilo da bandeira |
| onChange      | (value: {flag: FLAG, description: string} ) => void | false | método que recebe o valor selecionado |

### Exemplo

```javascript
import { Provider } from '@pismo/bolt-core'
import { FlagPanel, FLAG } from '@pismo/bolt-flag-panel'
```


<Playground>
    {
        () => {
            const data = [
                {flag: FLAG.br, description: 'Português'},
                {flag: FLAG.us, description: 'Inglês'},
                {flag: FLAG.es, description: 'Espanhol'}
            ]
            const [current, setCurrent] = useState(data[0])

            const handleChange = (value) => {
                setCurrent(value)
            }
            
            return (
                <Provider>
                    <FlagPanel
                        title='idioma'
                        data={data}
                        value={current}
                        onChange={handleChange}
                    />
                </Provider>
            )
        }
    }
</Playground>
