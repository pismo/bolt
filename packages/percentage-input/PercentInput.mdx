# PercentInput

O componente **PercentInput** fornece um componente TextField formatado para valores porcentagem, pronto para trabalhar com diversas condições de porcentagens em diferentes idiomas.

## Propriedades

| **prop**       | **type**                                                                   | **required** |   **default**   | **desc**                                                                         |
| -------------- | -------------------------------------------------------------------------- | :----------: | :-------------: | ---------------------------------------------------------------------------------|
| initialValue   | number                                                                     |    false     |        0        | valor inicial do componente                                                      |
| onChange       | (value: number) => void                                                    |    false     |      null       | evento disparado na mudança de valor do componente                               |
| TextfieldProps | [StandardTextFieldProps](https://material-ui.com/pt/api/text-field/#props) |    false     |       {}        | propriedades aplicadas ao componente TextField                                   |
| maxInteger     | number                                                                     |    false     |        0        | determina valor maximo                                                           |
| minInteger     | number                                                                     |    false     |        0        | determina valor minimo                                                           |
| toggleScore    | boolean                                                                    |    false     |      false      | substitui ponto por vírgula (* hiddenDecimal deve estar na condição false)       |
| hiddenDecimal  | boolean                                                                    |    false     |      false      | retira casas decimais padrao(.00) arredondando para cima ex: 0.5 igual a 100%    |
