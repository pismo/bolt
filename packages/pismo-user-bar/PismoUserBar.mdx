# PismoBar

Fornece um botão com dados do usuário fornecido, como e-mail, nome e avatar. O botão abre um Drawe container que pode ser populado com um componente livre

## Propriedades

| **prop**     | **type**        | **required** | **desc**                                                        |
| ------------ | --------------- | :----------: | --------------------------------------------------------------- |
| name         | string          |     true     | nome do usuário                                                 |
| email        | string          |     true     | e-mail do usuário                                               |
| notification | boolean         |    false     | torna um ponto visível sobre o avatar indicando uma notificação |
| src          | string          |    false     | imagem do avatar do usuário                                     |
| full         | boolean         |    false     | força o formato completo mesmo no mobile                        |
| contract     | boolean         |    false     | força o formato contraído, mesmo não sendo mobile               |
| children     | React.ReactNode |    false     | condeúdo do Drawer                                              |
